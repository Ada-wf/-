overview: >-
  全栈开发工程师是能同时驾驭前端和后端开发的"全能型"程序员。他们既能写出精美的用户界面，也能设计稳健的数据库结构和 API。从一个小型 SaaS 产品的 MVP 到企业级应用的某个功能模块，全栈工程师可以在技术栈的各层自由切换，独立完成从数据库到用户界面的完整开发链路。


  在创业公司和中小团队中，全栈工程师的价值尤为突出——一个人可以顶前端 + 后端两个人的产出。虽然每个层面的深度可能不如专项工程师，但他们的大局观和跨层调试能力让系统级的 bug 无所遁形。全栈工程师通常是技术团队的天然 leader 候选。
responsibilities:
  - '**全栈开发**: 同时负责前端界面和后端 API 的开发和维护'
  - '**数据库设计**: 设计数据库表结构，编写查询优化'
  - '**系统架构**: 参与系统架构设计，负责模块划分和技术选型'
  - '**部署运维**: 配置 CI/CD 流水线，管理云服务和服务器'
  - '**代码审查**: 在前端、后端和数据库层面审查同事的代码'
  - '**技术攻坚**: 解决跨层级的性能问题和系统 bug'
