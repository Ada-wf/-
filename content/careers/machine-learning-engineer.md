overview: >-
  机器学习工程师是 AI 落地的关键角色。他们不仅懂模型理论，更擅长将训练好的模型部署到生产环境，让模型能够在海量请求下稳定运行。从推荐系统的 A/B 实验到自动驾驶模型的端侧推理优化，从 NLP 模型的 API 化到计算机视觉模型的 GPU 集群调度——机器学习工程师站在算法研究和工程实践的交汇点上。


  与纯算法研究员不同，机器学习工程师更看重工程化能力：数据管道的搭建、模型的压缩和量化、推理服务的延迟优化、线上模型的持续监控和更新。他们需要同时和算法团队、后端团队和运维团队紧密合作。
responsibilities:
  - '**模型工程化**: 将训练好的模型转换为可规模部署的服务'
  - '**数据管道**: 搭建从数据采集、清洗、特征提取到模型训练的完整 pipeline'
  - '**模型优化**: 进行模型压缩、量化、蒸馏等优化以降低推理成本'
  - '**实验平台**: 搭建 A/B 实验和模型评估平台'
  - '**线上监控**: 监控模型在生产环境的表现，及时发现数据漂移问题'
  - '**框架开发**: 开发和维护内部 ML 框架和工具'
